SharpIni.Key.Key ( string  name = null,
string  value = null 
)

Constructs the instance with an optional name and value.

If the given name is invalid, Name will be set to string.Empty (See Naming.IsValid(string)).

If the given value is null, Value will be set to string.Empty (See Naming.IsValid(string)).
